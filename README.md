# Rahul's Portfolio

Personal portfolio website built using React, TypeScript, and Tailwind CSS.

## Tech Stack
- React (Vite)
- TypeScript
- Tailwind CSS
- Framer Motion
- Shadcn UI

## Features
- Responsive design
- Smooth animations
- Project showcase
- Resume download
- Contact form

## Setup

npm install
npm run dev

## Deployment
Deployed using Vercel
